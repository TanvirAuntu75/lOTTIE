# 🚀 SaaS Lottie Animation Pack v1.0

A professional collection of 8 animated icons designed for modern SaaS applications.

## 📦 What's Included
- **Checkmark** — Snappy success animation
- **Bell** — Gentle swing alert
- **Upload** — Upward arrow motion
- **Search** — Rotating glass perspective
- **Gear** — Continuous rotation
- **Heart** — Pulsing heartbeat effect
- **Star** — Spinning glow
- **Lock** — Secure closing action

## 🛠 Technical Details
- **Format:** Lottie JSON
- **Framerate:** 60 FPS
- **Resolution:** 500×500 (Scalable)
- **Palette:** Modern Indigo & Deep Purple

## 🌐 How to Use
1. Download `.json` files from `assets/`
2. Use [Lottie-Web](https://github.com/airbnb/lottie-web) or any Lottie player
3. Import into your web or mobile project
